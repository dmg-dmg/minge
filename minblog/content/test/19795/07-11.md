### 知识分享
1.初始化⼀一个Git仓库，使⽤
git init命令。
2.添加⽂文件到Git仓库，分两步：
  • 第⼀步，使⽤命令git add .注意，可反复多次使⽤用，添加多个⽂文件； 
  • 第二步，使用命令git commit，完成。
3.查看状态 
$ git status
4.撤销掉暂存区的修改
git reset HEAD file
5.文件删除
git rm (filename)
git commit -m 
6.丢弃⼯工作区的修改时
git checkout -- file
7.把本地库的所有内容推送到远程库上
$ git push -u origin master
8.创建dev分⽀
$ git checkout -b dev
