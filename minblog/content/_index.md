# 首页

1、按照类别，分别在`backend`,`po`,`front-end`,`test` 文件夹下创建自己工号对应的文件夹。

2、创建成功以后，在文件夹下创建`_index.md` 文件。文件内容如下：

``` markdown
+++
title = "工号（姓名）"
weight = 5
+++

# 这里是我的每日总结

* [日期](日期链接)
```

3、然后在自己的文件夹下按照日期创建文件。如：`07-10.md`。文档格式如下：

``` markdown
+++
title = "2018-07-10"
weight = 100
+++
```
其中`weight` 随着时间增加从100开始递减。
